# Ember chess engine
A chess engine written in C++, still in development. I'm making this in attempts to learn programming and to create a chess Bot which i'm really interested in.
Thanks to [Chess Programming Wiki](https://www.chessprogramming.org/Main_Page) for having been providing useful resources and knowledges.

# Techniques
## Move generation
- Bitboard chess representation
- Magic bitboard to generate sliding pieces moves

## Evaluation
- Material value comparison
- Piece-Square tables

## Search
- Alpha-Beta search algorithm

# Current performance
- 27 million NPM move generation speed
